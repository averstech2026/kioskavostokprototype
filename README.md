# A Vostok — Киоск самообслуживания

Прототип интерфейса киоска самообслуживания для столовой / фуд-корта (бренд [A Vostok](https://a-vostok.ru)).

## Возможности

- Стартовый экран с призывом приложить карту
- Каталог товаров по категориям с горизонтальной прокруткой
- Карточка товара с составом и пищевой ценностью
- Корзина с изменением количества
- Экран оплаты (дотация / банковская карта)
- Upsell: предложение добавить соус или пакет

## Запуск

```bash
npm install
npm run dev
```

Для демо в локальной сети:

```bash
npm run dev:lan
```

Сборка:

```bash
npm run build
npm run preview
```

## Стек

- HTML, CSS, vanilla JavaScript
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/) (CDN)
