Фото товаров с Unsplash

Устарело. Запустите npm run dev:images и откройте product-images.html.

Unsplash открывается в браузере, но не из терминала. Эта страница скачает фото в public/images.

  1. Убедитесь, что VPN включён (unsplash.com открывается).
  2. Запустите npm run dev и откройте http://localhost:5173/download-products.html.
  3. Нажмите кнопку и выберите папку public/images.
